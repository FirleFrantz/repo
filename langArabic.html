How to Install?  

• Locate your device below.

• Save the image to your device.

• Save/move the image to whichever theme folder you're using. Example:

MacBook Gray - /Library/Laetus/Themes/MacBook Gray/Images/

• Rename the image, using the names below. Example:

KeyboardLettersKeyPlanePortrait-896h@3x.png

This will replace the current default image for this layout. You won't be able to use multiple language layouts. (Until I can find a way around this).
