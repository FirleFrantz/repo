Shortcuts for Depths

Instructions:

Depths Downloader

Download this shortcut and run it to download the required images.

(This will save 6 images to iCloud/Shortcuts)

Add Depths

After running 'Depths Downloader', run 'Add Depths' shortcut.

This will overlay icon shadows on any wallpaper you choose.

(Includes an option to add a dock too.)

Please note:

This shortcut was made and tested only on an iPhone 11. It should work for notch devices only, but not for Max devices. *May* not work for iPhone X, or Mini iPhones.

This is the first version of this shortcut and may face issues I haven't noticed on my device. If you notice any issues, please reach out to me on Twitter: @Polyphian
